# TESTE BRASILPREV

Este serviço é responsável por realizar jogadas em um jogo fictício.


### Clone o Repositório:
```shell script
git clone git@github.com:victorfarruda/brasilprev.git
```

### Configurando projeto


Entre na pasta do projeto:
```shell script
cd brasilprev/
```

Crie um novo ambiente virtual ative-o e utilize o comando:
```shell script
pip install -r requirements.txt
```


Para rodar o script do jogo:
```shell script
python main.py
```

Para rodar os testes quando estamos com o ambiente virtual ativo, você pode usar o comando:
```shell script
pytest --cov . 
```
